## How To


### Save your changes

To save your changes use the `save.sh` script in the terminal. 

```
scripts/save.sh "This is version 1.2.x"
```

This command will store all changed documents and uploade them to Github.


### Load updates

To load other changes into your computer, use the `update.sh` script in the terminal

```
scripts/update.sh
```

This command will load all external changes into your computer.
